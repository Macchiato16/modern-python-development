# Modern Python Development Skill

A Codex skill for building, reviewing, and maintaining Python projects with modern engineering practices. It covers `src` layout, `pyproject.toml`, `uv`, Ruff, mypy, pytest, coverage, pre-commit, Git workflow, and GitHub Actions CI.

## Install

Clone this repository into your Codex skills directory:

```bash
git clone https://github.com/Macchiato16/modern-python-development.git ~/.codex/skills/modern-python-development
```

On Windows PowerShell:

```powershell
git clone https://github.com/Macchiato16/modern-python-development.git "$env:USERPROFILE\.codex\skills\modern-python-development"
```

Restart Codex after installing, then invoke the skill with `$modern-python-development`.
